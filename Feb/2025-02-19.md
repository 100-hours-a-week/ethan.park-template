## 날짜: 2025-02-19

### 스크럼
- 학습 내용 : 프로그래밍 패러다임
- 학습 내용 : 순수 함수

### 새로 배운 내용
- 프로그래밍 패러다임 : 프로그램을 구축하는 데 사용되는 기본적인 접근 방식
  - 대표적 프로그래밍 패러다임
    1. 절차적 프로그래밍 : 절차(순차)적인 명령 수행뿐만 아니라, 프로시저 호출의 개념을 바탕으로 하고 있는 프로그램 패러다임
    2. 객체지향 프로그래밍 : 최소 단위가 객체이며, 객체를 만들고, 이 객체를 사용하여 데이터와 메소드를 묶고 소통하는 프로그래밍
    3. 함수형 프로그래밍 : 최소 단위가 함수이며, 함수를 통해 새로운 데이터들을 만들어가며 파이프 형태로 프로그래밍
- 순수 함수 : 주어진 입력이 동일할 때 항상 같은 값을 리턴하는 함수
  - 특징
    1. 일관된 결과 : 동일한 입력 값에 대해 항상 동일한 결과 반환
    2. 부수 효과 없음 : 함수 외부의 상태를 변경하지 않으며, 외부 변수에 의존하지 않는다.
    3. 명확한 구성 요소 : 파라미터(입력), 로직(처리), 반환(출력)이 명확하게 정의
